Initialise the L\+C\+D and setup the \hyperlink{structlcd__display}{lcd\+\_\+display} structure. 

Uses the configuration to set up the display in a default state. When method returns, display is not on and will need to be activated with \hyperlink{lcd_8h_ab643ebf351f45a4ebd8e9c38902919a7}{lcd\+\_\+display\+On()}. This allows you to perform any further configuration tasks at runtime before the display is enabled or to prepare the display at an earlier stage to activate later.

Displays a mono bitmap to a given region. 

For each pixel in the region (going from left to right before wrapping around) one bit of the data will be read (from L\+S\+B to M\+S\+B) and if that bit is a {\ttfamily 1} forground will be displayed and if it is a {\ttfamily 0} background will be used. Foreground and background are based on that defined in the {\ttfamily \hyperlink{structlcd__display}{lcd\+\_\+display}}.

{\ttfamily bpr} defines how many bits should be used per row. This allows for better use of the available bits when the width of the region is not divisible by 8. When the width of the region is less than the {\ttfamily bpr} value, then background will be writen into the rest of the given region\textquotesingle{}s width. When {\ttfamily bpr} is greater than the width of the given region, additional bits of data will be ignored.
